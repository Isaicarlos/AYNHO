# AYNHO
Modelado de la entropía mental y dinámica de sistemas complejos en la cognición"
AYNO es una iniciativa que busca unificar el formalismo de la Física Estadística y la Teoría de la Información para modelar la dinámica cerebral en condiciones de incertidumbre. El proyecto se centra en la "mente como campo de batalla", entendida como una competencia estocástica entre estados de orden y caos cognitivo.
